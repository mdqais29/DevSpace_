# DevSpace – AI Developer Workspace

A modern, responsive, and SaaS-inspired developer workspace built with **React** and **Vite**. DevSpace features a clean user interface, interactive dashboard, responsive layouts, and a professional design inspired by modern AI productivity platforms.

🌐 **Live Demo:** https://devspaceproject.netlify.app

---

##  Features

- Modern SaaS-inspired UI
- Fully Responsive Design
- Built with React + Vite
- Multi-page Navigation using React Router
- Interactive Dashboard
- Profile & Settings Pages
- Dark Theme Interface
- Analytics Cards
- AI Workspace Layout
- Reusable React Components
- Fast Performance with Vite

---

##  Tech Stack

- React
- Vite
- React Router DOM
- JavaScript (ES6+)
- HTML5
- CSS3
- TypeScript

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/mdqais29/DevSpace_.git
```

### Navigate into the Project

```bash
cd DevSpace_
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
